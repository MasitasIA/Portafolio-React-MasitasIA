Este es el código fuente de mi portafolio personal, diseñado para mostrar mi experiencia, habilidades y proyectos destacados en el desarrollo de software. Es una Single Page Application (SPA) construida con un enfoque en el rendimiento, la accesibilidad y el diseño responsivo.

## ✨ Características

- **Diseño Responsivo:** Adaptable a dispositivos móviles, tablets y escritorio.
- **Modo Oscuro Integrado:** Paleta de colores personalizada (Zinc) para una experiencia visual moderna y elegante.
- **Navegación Fluida:** Scroll suave (`smooth scroll`) entre las distintas secciones de la página.
- **Menú Móvil Interactivo:** Navegación optimizada para pantallas pequeñas (Menú Hamburguesa).
- **Tipografía Personalizada:** Integración de fuentes locales para los títulos.

## 🛠️ Tecnologías y Herramientas

- **Framework:** [React 18](https://react.dev/)
- **Entorno de Desarrollo:** [Vite](https://vitejs.dev/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Iconos:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)

## 📦 Dependencias del Proyecto

El proyecto está construido y configurado con las siguientes librerías principales, definidas en el archivo `package.json`:

### Dependencias Principales
* **[React](https://react.dev/) & React DOM (`^19.2.0`):** Librería core para la construcción de la interfaz de usuario.
* **[TailwindCSS Vite Plugin](https://tailwindcss.com/) (`^4.2.1`):** Integración oficial de Tailwind v4 directamente en el empaquetador.
* **[Lucide React](https://lucide.dev/) (`^0.575.0`):** Colección de iconos SVG minimalistas y adaptables para la interfaz.
* **[React Icons](https://react-icons.github.io/react-icons/) (`^5.5.0`):** Biblioteca para importar los logotipos oficiales de las tecnologías mostradas en la sección de habilidades.

### Dependencias de Desarrollo (Destacadas)
* **[Vite](https://vitejs.dev/) (`^7.3.1`):** Entorno de desarrollo local ultrarrápido y empaquetador para producción.
* **[TypeScript](https://www.typescriptlang.org/) (`~5.9.3`):** Añade tipado estático al proyecto para un código más robusto y sin errores.
* **[Tailwind CSS](https://tailwindcss.com/) (`^4.2.1`):** Motor principal para la generación de clases de estilos.
* **[ESLint](https://eslint.org/) (`^9.39.1`):** Herramienta de linting configurada con múltiples plugins (`eslint-plugin-react-hooks`, `typescript-eslint`) para asegurar la calidad y limpieza del código.
